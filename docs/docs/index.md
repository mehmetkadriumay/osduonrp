# OSDU CI Butler

The CI Butler is an easy to use full CLI for installing OSDU locally.  In short, CIButler is a tool for development, automation, testing and interacting with OSDU directly. It was built to make installing Community Implementation of OSDU easier and faster with OSDU.

The CI Butler is supported on Mac, Windows and Linux.

CI Butler also have built-in support for managing users and client ids in keycloak and adding users to groups OSDU.
This can be especially useful to when you need to redeploy your OSDU environment or test as part of a pipeline.

CI Butler has built-in help:

- `cibutler --help`
- `cibutler command --help`
